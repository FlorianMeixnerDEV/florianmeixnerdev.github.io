Alan Walker Music - Wiki

Introduction to the Mod

The Alan Walker Music Mod is a mod about the DJ and songwriter Alan Walker.

The core of the mod is a custom music disc featuring some of his most famous songs.

Additional features and mobs are described below.

1. Custom Music-Disc

This mod includes special music discs with Alan Walker tracks:

  • Avalon
  • Believers
  • Creator Circle
  • Dust
  • Faded
  • Incommunicado
  • Moonshine
  • Old Habits
  • Time
  • Welcome To Walkerworld
  • Rise of the Drones
  • Sailing
  • Welcome to Lonely Club
  • ...and more

2. Custom Mobs

Three unique mobs can be summoned in your world:

  • Alan Walker: Peaceful
    Can be tamed with a diamond.
  • Bad Alan: Aggressive
    Mob with red eyes.
  • The Guard: Protective
    Protects you from evil mobs.

3. Installation

Method 1: Manual Installation
  1. Ensure Minecraft & NeoForge 1.21.8 are installed (link).
  2. Download the Alan Walker Music Mod .jar file.
  3. Place it in your Minecraft "mods" folder.
  4. Launch Minecraft with NeoForge to load the mod.
Method 2: Via Modrinth Launcher

You can also install it using the Modrinth Launcher. Simply search for "Alan Walker Music" and follow the prompts.

4. Commands

Use the following commands in-game:

  • /summon alan_walker_music:alan_walker
    Summons Alan Walker mob.
  • /summon alan_walker_music:bad_alan
    Summons Bad Alan mob.
  • /summon alan_walker_music:the_guard
    Summons The Guard mob.
  • /info
    Display website info.

Installation Overview
  1. Extract the downloaded ZIP file.
  2. Place the extracted folder in your Minecraft "saves" directory.
  3. Launch Minecraft and select the map from the "Singleplayer" menu.

This guide applies only to the Modrinth launcher. Steps may differ for Mojang or other third-party launchers.

  1. Open the Minecraft folder by typing %appdata% in Windows search and press Enter, then navigate to ".minecraft" folder.
  2. Open the "saves" folder.
  3. Unzip the downloaded ZIP file and drag the folder into your "save" folder.

Preparation
  • Download the desired Minecraft map file (.zip)
  • Extract the .zip file
  • Important: Ensure the extracted folder contains the level.dat file directly (not in a subfolder)
Step 1: Open CurseForge Folder
  1. Open the CurseForge Launcher
  2. Right-click on the modpack profile in "My Modpacks" where you want to play the map
  3. Select "Open Folder" to access the modpack directory
Step 2: Add the Map
  1. Navigate to the "saves" folder in the opened directory
  2. If no "saves" folder exists, create one with that name
  3. Move the extracted map folder into the "saves" folder
Step 3: Launch Game
  1. Start the modpack through CurseForge
  2. The new world should now be available in the Singleplayer menu
Important Note: Ensure the map is compatible with the Minecraft version used by your modpack.
