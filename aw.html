Introduction to the Mod

The Alan Walker Music Mod is a mod about the DJ and songwriter of the same name, Alan Walker.

The core of the mod is a custom music disc created specifically for this mod, featuring well-known songs by Alan Walker.

Furthermore, we have included some small features, which are described below.

2. Custom Music-Disc

The mod includes its own discs. These contain some songs produced by Alan Walker.

Songs:

  • Avalon
  • Old Habits
  • WorldOfWalker - Rise of the Drones
  • Believers
  • Creator Circle
  • Dust
  • Faded
  • Heartbreak Melody
  • Time
  • Moonshine
  • Incommunicado
  • Welcome to Walkerworld
  • Sailing
  • Welcome to Lonely Club

3. Custom Mobs

The mod contains custom mobs. The following mobs are included:

  • Alan Walker:

    This mob is peaceful. You can "tame" it with a diamond, and it will then follow you everywhere.

  • Bad Alan:

    This mob is recognizable by its red eyes. It attacks you and inflicts damage.

  • The Guard:

    Its protects you from all Evil Mobs

4. Installation

To install the Alan Walker Music Mod, follow these steps:

  1. Ensure you have Minecraft and NeoForge 1.21.8 installed. Installer found here
  2. Download the Alan Walker Music Mod .jar file from a trusted source.
  3. Locate your Minecraft "mods" folder. This is usually found in the ".minecraft" directory.
  4. Copy the downloaded .jar file into the "mods" folder.
  5. Launch Minecraft with the NeoForge profile to load the mod.

Installation over Modrinth

Alternatively, you can use the Modrinth Launcher to easily install and manage the Alan Walker Music Mod along with other mods. Simply search for "Alan Walker Music Mod" within the launcher and follow the prompts to install it.
