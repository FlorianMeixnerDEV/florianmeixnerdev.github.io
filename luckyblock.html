Introduction to the Mod

The LuckyBlock Mod adds exciting random events to Minecraft by introducing special blocks that trigger various effects when broken.

The core of the mod is the Lucky Block, which can give you items, spawn mobs, or cause fun and surprising events.

Furthermore, we have included some custom effects, which are described below.

2. Lucky Blocks

The mod includes various lucky blocks that provide random rewards or effects.

Possible Effects:

  • Spawn Diamonds
  • Summon Creepers
  • Give Random Items
  • Teleport Player
  • Spawn TNT
  • Heal Player
  • Spawn Animals
  • Give Potions
  • Build Structures
  • Spawn Monsters
  • Random Weather
  • Enchant Items
  • Spawn Fireworks

    • and more

3. Custom Effects

The mod contains custom effects for lucky blocks. The following effects are included:

  • Explosion:

    Creates a small explosion around the block.

  • Rain Diamonds:

    Drops diamonds from the sky.

  • Spawn Mob Army:

    Spawns a group of friendly mobs to help you.

4. Installation

To install the LuckyBlock Mod, follow these steps:

  1. Ensure you have Minecraft and NeoForge 1.21.8 installed. Installer found here
  2. Download the LuckyBlock Mod .jar file from a trusted source.
  3. Locate your Minecraft "mods" folder. This is usually found in the ".minecraft" directory.
  4. Copy the downloaded .jar file into the "mods" folder.
  5. Launch Minecraft with the NeoForge profile to load the mod.

Installation over Modrinth

Alternatively, you can use the Modrinth Launcher to easily install and manage the LuckyBlock Mod along with other mods. Simply search for "LuckyBlock" within the launcher and follow the prompts to install it.
